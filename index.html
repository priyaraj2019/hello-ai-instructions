Why repository instructions matter

Repository instructions help GitHub Copilot understand your project's context, coding standards, and expectations. This creates a personalized development experience where Copilot gives you advice that fits your specific use case.

This practice repo keeps everything simple—just plain HTML and CSS—so you can focus on learning how Copilot instructions work without framework complexity.

Get started

  1. Check the .github/copilot-instructions.md file to see repository guidelines
  2. Ask GitHub Copilot to help improve this page or suggest changes
  3. Notice how Copilot respects the beginner-friendly, no-framework approach
